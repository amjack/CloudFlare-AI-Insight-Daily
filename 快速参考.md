## 🔔 配置 Webhook 通知

### GitHub Actions

1. **添加 Secret：** `Settings` → `Secrets` → `New secret`
   - Name: `WEBHOOK_URL`
   - Value: 你的 webhook 地址

2. **添加 Variable：** `Settings` → `Variables` → `New variable`
   - Name: `WEBHOOK_TYPE`
   - Value: `dingtalk` / `feishu` / `wecom` 等

3. **（可选）自定义 GitHub Pages URL：**
   - Name: `GITHUB_PAGES_URL`
   - Value: `https://你的域名` 或 `https://你的用户名.github.io/仓库名/`

### Docker

在 `Dockerfile` 中设置：

```dockerfile
ENV WEBHOOK_URL="你的webhook地址"
ENV WEBHOOK_TYPE="dingtalk"
ENV GITHUB_PAGES_URL="https://你的页面地址"
```

## ⏰ 修改执行时间

### GitHub Actions

编辑 `.github/workflows/build-daily-book.yml`：

```yaml
schedule:
  - cron: '0 1 * * *'  # UTC 01:00 = 北京时间 09:00
```

**时间对照：**
- 北京 08:00 = `0 0 * * *`
- 北京 09:00 = `0 1 * * *` ✅ 当前
- 北京 10:00 = `0 2 * * *`
- 北京 20:00 = `0 12 * * *`

### Docker

编辑 `Dockerfile`：

```dockerfile
RUN echo "0 9 * * * /app/scripts/build.sh ..." > /etc/crontabs/root
#         ↑ 改这里（北京时间）
```

## 🛠️ 常用命令

### Git 操作

```bash
# 创建 diy 分支
git checkout -b diy
git push -u origin diy

# 切换分支
git checkout diy
git checkout main

# 查看所有分支
git branch -a
```

### Docker 操作

```bash
# 构建镜像
docker build -t ai-daily-cron-job .

# 运行容器
docker run -d \
  --name ai-daily-cron \
  -p 4399:4399 \
  -e WEBHOOK_URL="你的webhook地址" \
  --restart always \
  ai-daily-cron-job

# 查看日志
docker logs ai-daily-cron

# 进入容器
docker exec -it ai-daily-cron /bin/sh

# 停止并删除
docker stop ai-daily-cron && docker rm ai-daily-cron
```

### GitHub Actions

```bash
# 查看 Actions：
# GitHub 仓库 → Actions 标签

# 手动触发：
# Actions → Build Daily Journal → Run workflow
```
