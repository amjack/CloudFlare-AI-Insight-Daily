## 🔄 如何生成真实内容？

1. **配置 Cloudflare Workers**
   - 部署 Workers 到 Cloudflare
   - 配置 Google Gemini API
   - 设置 Folo Cookie

2. **配置 GitHub Secrets**
   - `WRITE_RSS_URL`: Workers 的 API 地址
   - `RSS_FEED_URL`: RSS Feed 地址

3. **等待自动更新**
   - 每天早上 9:00 自动运行
   - 或手动触发 GitHub Actions
