# 🎙️ 播客脚本

> 🎵 AI 资讯播客口播稿

## 📻 关于播客

AI 资讯日报不仅提供文字版的日刊，还可以生成适合播客的口播稿。

### 播客特点

- 🎯 **结构化脚本**：开场、正文、结尾完整结构
- 🎨 **口语化表达**：适合语音播报的流畅表达
- ⏱️ **时长估算**：提供预估播报时长
- 📝 **重点标记**：标注关键信息和停顿位置

## 🎬 播客脚本示例

```
【开场白】
大家好，欢迎来到 AI 资讯日报播客版。我是你们的 AI 主播。
今天是2024年12月2日，让我们一起来看看 AI 领域有哪些值得关注的动态。

【正文】
首先是今天的头条新闻...
（这里将展示 AI 生成的播客脚本）

【结尾】
以上就是今天的 AI 资讯播客，感谢收听！
我们明天同一时间再见。
```

## 🔧 如何生成播客脚本？

播客脚本通过以下流程自动生成：

1. **日刊内容生成**
   - Cloudflare Workers 获取并整理 AI 资讯

2. **AI 格式化**
   - 使用 Google Gemini 将文字转换为口播稿
   - 优化语言表达，适合语音播报

3. **自动发布**
   - 保存到 podcast/ 目录
   - 构建到 mdBook 网站
